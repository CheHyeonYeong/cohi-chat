# Issue #78: FE 로그인 API 마이그레이션 (useLogin.ts)

## 목표
백엔드를 Python FastAPI에서 Spring Boot로 전환함에 따라 로그인 API 엔드포인트 변경

## API 변경 사항

| 구분 | 변경 전 | 변경 후 |
|------|--------|--------|
| 경로 | `POST /account/login` | `POST /members/v1/login` |
| 요청 | `{username, password}` | `{username, password, provider}` |
| 응답 | `{accessToken, tokenType, user}` | `{accessToken, expiredInMinutes, refreshToken, username, displayName}` |
