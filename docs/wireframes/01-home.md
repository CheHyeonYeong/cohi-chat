## UI 요소 설명

### 1. 헤더 영역
- **타이틀**: "약속 잡기 서비스"
- **로그아웃 버튼**: 로그인 상태에서만 표시 (우측 상단)

### 2. 호스트 목록
- 각 호스트는 카드 형태로 표시
- **표시 정보**: 표시 이름 (username)
- **동작**: 클릭 시 해당 호스트의 캘린더 페이지로 이동
- **스타일**: 호버 시 배경색 변경, 화살표 아이콘으로 클릭 가능 표시

## 상태별 처리

| 상태 | 동작 |
|------|------|
| 로딩 중 | "로딩 중..." 메시지 표시 |
| 에러 (401) | "로그인이 필요합니다" 메시지 + 로그인 버튼 |
| 에러 (기타) | 에러 메시지 표시 |
| 성공 | 호스트 목록 표시 |

## 인터랙션

1. **호스트 카드 클릭** → `/calendar/{slug}` 페이지로 이동
2. **로그아웃 버튼 클릭** → 로그아웃 처리 후 페이지 새로고침
