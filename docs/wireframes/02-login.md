## UI 요소 설명

### 1. 타이틀
- "로그인" 텍스트
- 중앙 정렬, 굵은 글씨

### 2. 입력 필드
| 필드 | 타입 | placeholder | 필수 |
|------|------|-------------|------|
| 아이디 | text | - | O |
| 비밀번호 | password | - | O |

### 3. 로그인 버튼
- **스타일**: Primary 버튼 (파란색 배경, 흰색 텍스트)
- **너비**: 전체 너비
- **비활성**: 로딩 중일 때

### 4. 회원가입 링크
- "계정이 없으신가요? 회원가입" 텍스트
- "회원가입" 부분은 링크 스타일 (밑줄)
- 클릭 시 `/app/signup`으로 이동

## 인터랙션

1. **폼 제출** → 유효성 검사 → API 호출 → 성공 시 `/app`으로 리다이렉트
2. **회원가입 링크 클릭** → `/app/signup`으로 이동

## 입력 유효성

- 아이디: 필수 입력
- 비밀번호: 필수 입력
