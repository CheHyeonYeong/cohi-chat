## UI 요소 설명

### 1. 타이틀
- "회원가입" 텍스트
- 중앙 정렬, 굵은 글씨

### 2. 입력 필드

| 필드 | 타입 | 필수 | 길이 제한 | 설명 |
|------|------|------|----------|------|
| 아이디 | text | O | 4-40자 | 로그인에 사용할 ID |
| 이메일 | email | O | 최대 128자 | 이메일 주소 |
| 표시 이름 | text | X | 4-40자 | 다른 사용자에게 보여질 이름 |
| 비밀번호 | password | O | 8-128자 | 비밀번호 |
| 비밀번호 확인 | password | O | 8-128자 | 비밀번호 재입력 |

### 3. 회원가입 버튼
- **스타일**: Primary 버튼 (파란색 배경, 흰색 텍스트)
- **너비**: 전체 너비
- **비활성**: 로딩 중일 때

### 4. 로그인 링크
- "이미 계정이 있으신가요? 로그인" 텍스트
- "로그인" 부분은 링크 스타일
- 클릭 시 `/app/login`으로 이동

## 인터랙션

1. **폼 제출** → 유효성 검사 → API 호출 → 성공 시 1.5초 후 `/app/login`으로 리다이렉트
2. **로그인 링크 클릭** → `/app/login`으로 이동

## 유효성 검사

| 필드 | 검사 항목 |
|------|----------|
| 아이디 | 4-40자, 필수 |
