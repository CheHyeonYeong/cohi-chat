## UI 요소 설명

### 1. 헤더 영역
- **첫 화면으로 버튼**: `/app`으로 이동
- **내 예약 목록 버튼**: `/app/my-bookings`로 이동
- **타이틀**: "{호스트 표시명}님과 약속잡기"

### 2. 네비게이터 (Navigator)
- **이전 달 버튼 (<)**: 이전 달로 이동 (과거는 비활성화)
- **현재 월 표시**: "2024년 2월"
- **다음 달 버튼 (>)**: 다음 달로 이동

### 3. 달력 본문 (Body)
| 요소 | 스타일 |
|------|--------|
| 요일 헤더 | 일~토, 중앙 정렬 |
| 예약 가능 날짜 | 파란색 배경 (`bg-blue-50`), 클릭 가능 |
| 예약 불가 날짜 | 회색 텍스트, 클릭 불가 |
| 선택된 날짜 | 파란색 테두리 |
| 날짜 셀 | 48x48px 원형 |

### 4. 시간대 영역 (Timeslots)
- 선택된 날짜의 예약 가능한 시간대 목록
- 각 시간대는 버튼 형태
- 클릭 시 예약 폼으로 전환

### 5. 예약 폼 (BookingForm)
| 필드 | 타입 | 설명 |
|------|------|------|
| 주제 | select | 드롭다운에서 선택 |
| 설명 | textarea | 추가 설명 입력 |

## 인터랙션

1. **월 네비게이션** → URL 쿼리 파라미터 업데이트
2. **날짜 클릭** → 시간대 목록 표시
3. **시간대 클릭** → 예약 폼 표시
4. **달력으로 돌아가기** → 시간대 선택 초기화
5. **예약 신청** → API 호출 → 성공 시 알림 표시
