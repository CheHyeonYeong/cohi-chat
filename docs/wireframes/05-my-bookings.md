## UI 요소 설명

### 1. 헤더
- **첫 화면으로 버튼**: `/`으로 이동

### 2. 타이틀
- "내 예약 목록" 텍스트
- 중앙 정렬, 굵은 글씨

### 3. 예약 카드
| 정보 | 설명 |
|------|------|
| 날짜 | YYYY.MM.DD (요일) |
| 시간 | HH:MM - HH:MM |
| 호스트 | {호스트명}님과 |
| 주제 | {주제} 주제로 예약 |

- **스타일**: 테두리, 라운드 코너, 호버 시 배경색 변경
- **클릭**: 예약 상세 페이지로 이동

### 4. 페이지네이션 (Pagination)
| 요소 | 설명 |
|------|------|
| 이전 버튼 | 이전 페이지로 이동 (첫 페이지에서 비활성) |
| 페이지 정보 | "1 / 3 페이지" 형식 |
| 다음 버튼 | 다음 페이지로 이동 (마지막 페이지에서 비활성) |

## 인터랙션

1. **예약 카드 클릭** → `/booking/{id}`로 이동
2. **이전 버튼 클릭** → 이전 페이지 데이터 로드
3. **다음 버튼 클릭** → 다음 페이지 데이터 로드
4. **로그인 버튼 클릭** → `/login`으로 이동

## 페이지네이션 설정

- **기본 페이지 크기**: 10개
- **URL 파라미터**: `?page=1`
