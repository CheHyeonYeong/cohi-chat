## UI 요소 설명

### 1. 헤더
- **내 예약 목록으로 버튼**: `/app/my-bookings`로 이동

### 2. 타이틀
- "{호스트명}님과 약속잡기" 텍스트
- 중앙 정렬, 굵은 글씨

### 3. 예약 정보 카드
| 정보 | 형식 |
|------|------|
| 주제 | 아이콘 + 주제명 |
| 날짜 | YYYY년 M월 D일 (요일) |
| 시간 | HH:MM - HH:MM |

### 4. 설명 영역
- 예약 시 입력한 설명 텍스트 표시
- 배경색으로 구분

### 5. 파일 첨부 영역
| 요소 | 설명 |
|------|------|
| 파일 선택 | 파일 입력 필드 (다중 선택 가능) |
| 첨부하기 버튼 | 파일 업로드 실행 |

### 6. 첨부된 파일 목록
| 요소 | 설명 |
|------|------|
| 파일 아이콘 | 📎 클립 아이콘 |
| 파일명 | 파일 이름 표시 |
| 다운로드 링크 | 클릭 시 파일 다운로드 |

## 인터랙션

1. **내 예약 목록으로 버튼 클릭** → `/app/my-bookings`로 이동
2. **파일 선택** → 파일 선택 다이얼로그 열기
3. **첨부하기 버튼 클릭** → 파일 업로드 API 호출
4. **다운로드 링크 클릭** → 파일 다운로드

## 파일 업로드 제한 (참고)

- **지원 형식**: 모든 파일 형식
- **다중 업로드**: 지원
- **업로드 후**: 목록에 즉시 반영
