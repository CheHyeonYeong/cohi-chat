## 디자인 시스템

### 색상

| 이름 | 색상 코드 | 용도 |
|------|----------|------|
| Primary | `#1A8DF7` | 주요 버튼, 강조 텍스트, 선택된 날짜 |
| Secondary | `#65BEFF` | 호버 효과, 보조 요소 |
| Error | `#F04E4E` | 에러 메시지 |
| Success | `#17BC6D` | 성공 메시지 |
| Warning | `#FE9922` | 경고 메시지 |
| Info | `#154093` | 정보 메시지 |

### 레이아웃

- **모바일**: 전체 화면 너비
- **태블릿/데스크톱**: 최대 800px 너비, 중앙 정렬
- **컨테이너**: 테두리, 라운드 코너, 내부 패딩 (px-4 py-8)

### 버튼 스타일

- **Primary**: 파란색 배경 (#1A8DF7), 흰색 텍스트
- **Secondary**: 회색 배경, 어두운 텍스트
